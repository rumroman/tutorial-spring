                                        XML.

XML - это промышленный стандарт, определенный W3C.Хотя он не привязан ни к какому языку программирования или поставщику
программного ообеспечения , этот язык решил проблему независимости данных и совместимости. XML являетмя расширяемым, не
зависит от платформы и поддерживает интернационализацию, поэтому он стал предпочтительным языком для обмена данными
между программными компонентами ,системами и предприятиями.
    С другой стороны JSON появился веместе с JavaScript и служит для представления простых структур данных менее
избыточным , нежели XML способом. Он быстро стал настолько популярным .

                                    Основные сведения об  XML.

XML - разрабатывался как язык для опредления новых форматов документов во Всемирной паутине.
W3C - это консорциум, который известен развитием и поддержанием веб-технологий , таких как HTML, XHTML, RDF или CSS.
W3C также является центральным органом XML и всех связанных с XML технологий .
